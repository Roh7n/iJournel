# Motion Director Skill

Guidelines for planning and implementing high-quality animations using `motion/react`.

## Purpose:

- Advanced animations
- Motion systems
- Scroll experiences

## Expertise:

- Motion
- View transitions
- UI Motion Design
- Advanced Interaction Design

## Rules:

- Animation serves content
- Avoid motion spam
- Define motion hierarchy
- Maintain 60fps

## Responsibilities:

- Page transitions
- Scroll animations
- Parallax systems
- Reveal systems
- Cursor effects
- Text animations
