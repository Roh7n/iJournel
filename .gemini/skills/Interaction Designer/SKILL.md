# Interaction Designer Skill

Best practices for managing interactive states, event handlers, and accessibility (a11y).

## Objectives

- Focus on hover effects, active states, buttons, and micro-interactions.
- Keep the interface interactive and alive.

## Purpose:

- Micro-interactions
- Hover states
- User feedback

## Responsibilities:

- Button states
- Card interactions
- Navigation behavior
- Cursor interactions
- Gestures
